1. PROJECT OVERVIEW

NetNynja Enterprise is a unified network management platform that consolidates three critical IT infrastructure management capabilities into a single, integrated solution. Designed for enterprise environments, it provides comprehensive visibility and control over network infrastructure while maintaining security compliance.

Mission Statement: To provide IT and security teams with a unified, secure platform for managing IP addresses, monitoring network performance, and ensuring security compliance across enterprise infrastructure.

Core Applications

Application Purpose Key Capabilities
IPAM
IP Address Management		 Centralized IP address tracking and network space management
  • Network discovery (ICMP, TCP, NMAP)
  • DHCP/DNS integration
  • Capacity planning and utilization reporting
  • OS fingerprinting (TTL-based)
NPM
Network Performance Monitoring		 Real-time monitoring of network device health and performance
  • SNMPv3 polling (FIPS-compliant: SHA-256+, AES-256)
  • Device discovery with vendor detection
  • CPU, memory, latency, interface metrics
  • Alerting with customizable thresholds
STIG Manager
Security Compliance		 Automated STIG compliance auditing and reporting
  • SSH/Netmiko remote auditing
  • CKL/CKLB/XML import and generation
  • Support for 16+ platforms (Cisco, Juniper, Palo Alto, etc.)
  • Compliance scoring and dashboards

Target Deployment Platforms

  • macOS (ARM64/x64) - Docker Desktop
  • Red Hat Enterprise Linux 9.x - Podman/Docker compatible
  • Windows 11 / Windows Server - Docker Desktop with WSL2
